# Practica 2 peer to peer

Repositorio para la practica 2 del curso de computación distribuida.<br>
Todos los comandos se ejecutan desde el directorio raiz de la práctica.

## Comando para compilar el cliente en Windows.

> javac -d peer/build peer/src/main/java/\*.java

## Comando para ejecutar el cliente en Windows.

> java -classpath peer/build Main PUERTO NOMBRE
